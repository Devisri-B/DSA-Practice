## How this repository will be organized

1. **Create a Problem File**: Create a new .py file for each problem in the appropriate topic folder
2. **Use Template**: Start with the template from `utils/template.py` as a foundation
3. **Implement Solution**: Write solution with clear comments and docstrings
4. **Add Test Cases**: Include test cases within the file
5. **Update Progress**: Update this README with your progress

## Topics Overview

### Arrays
Foundation of DSA. Master array manipulation, searching, sorting, and optimization techniques.

### Binary Search
Efficient searching in sorted arrays. Learn different binary search patterns and applications.

### Linked Lists
Non-contiguous data structures. Practice pointer manipulation and linked list operations.

### Trees
Hierarchical data structures. Learn traversals, BST operations, and tree algorithms.

### Graphs
Network and relationship modeling. Master graph traversals, shortest paths, and connectivity.

### Dynamic Programming
Optimization technique for overlapping subproblems. Build up solutions from smaller sub-problems.
